еҝ«йҖҹдҪҝз”Ё Firebase Cloud Functions еүөе»әз¶Ій ҒеҫҢз«ҜAPI

Firebase Cloud Functions д»Ӣзҙ№

Firebase Cloud Functions жҳҜ Firebase зҡ„дёҖй …жңҚеӢҷпјҢиғҪеӨ и®“дҪҝз”ЁиҖ…еңЁдёҚйңҖиҰҒиҖғж…®дјәжңҚеҷЁзҡ„жғ…жіҒдёӢпјҢе»әз«ӢеҸҜд»ҘеңЁйӣІз«Ҝеҹ·иЎҢзҡ„зЁӢејҸзўјпјҢдёҰдҪҝз”ЁHTTPSжҲ–е…¶д»–Firebaseзҡ„жңҚеӢҷйҖІиЎҢиӘҝз”ЁпјҢдҪҶжҳҜиӢҘиҰҒдҪҝз”ЁйҖҷй …жңҚеӢҷпјҢеҝ…й Ҳе°Үе°ҲжЎҲеҚҮзҙҡжҲҗBlaze PlanжүҚиғҪеӨ дҪҝз”ЁпјҢдёҰдё”иӘҝз”ЁжҳҜйңҖиҰҒиІ»з”Ёзҡ„пјҢдҪҶжҳҜжҜҸеҖӢжңҲзҡ„е…ҚиІ»йЎҚеәҰе°ҚдёҖиҲ¬зҡ„й–ӢзҷјиҖ…дҫҶиӘӘжҮүи©ІжҳҜзӣёз•¶еӨ з”ЁдәҶ(дёҖеҖӢжңҲ200иҗ¬ж¬Ўе‘јеҸ«)пјҢиІ»з”Ёи©іжғ…еҸҜд»ҘеҸғиҖғйҖҷиЈЎпјҢйҮқе°Қи«ӢжұӮж•ёгҖҒе„Іеӯҳз©әй–“гҖҒеҹ·иЎҢжҷӮй–“зӯүйғҪжңүи©ізҙ°зҡ„еғ№ж јиӘӘжҳҺгҖӮ

й–Ӣе§ӢдҪҝз”Ё Firebase Cloud Functions

еңЁдҪҝз”Ёд»»дҪ• Firebase зӣёй—ңзҡ„жңҚеӢҷжҷӮпјҢйңҖиҰҒе…Ҳе»әз«ӢдёҖеҖӢе°ҲжЎҲе’Ңз¶Ій ҒжҮүз”ЁзЁӢејҸпјҢйҒёж“ҮвҖқж–°еўһе°ҲжЎҲвҖқеҫҢпјҢи«ӢжҢүз…§жөҒзЁӢеҫҖдёӢиө°жҠҠе°ҲжЎҲе»әз«Ӣиө·дҫҶгҖӮ

жҺҘи‘—пјҢй»һйҒёз¶Ій Ғе»әз«Ӣз¶Ій ҒжҮүз”ЁзЁӢејҸпјҢжҺҘи‘—ијёе…ҘжҮүз”ЁзЁӢејҸеҗҚзЁұпјҢдҫқз…§жөҒзЁӢи·‘е®ҢгҖӮ

и«ӢиЁҳеҫ—е°Үе°ҲжЎҲеҚҮзҙҡиҮіBlaze Plan

е®үиЈқ Firebase CLI

иҰҒдҪҝз”ЁFirebase FunctionsйӮ„йңҖиҰҒеңЁжң¬ең°е®үиЈқ Firebase CLI пјҢйҖҷиЈЎдҪҝз”ЁNPMзҡ„еҪўејҸеҒҡе®үиЈқгҖӮ

е®үиЈқжҲҗеҠҹеҫҢпјҢйңҖиҰҒеҒҡиә«еҲҶй©—иӯүпјҢи«ӢйҖҸйҒҺд»ҘдёӢжҢҮд»Өзҷ»е…Ҙ

зҷ»е…ҘжҲҗеҠҹд№ӢеҫҢпјҢи«ӢдҪҝз”Ёд»ҘдёӢжҢҮд»ӨзўәиӘҚеёіжҲ¶е…§жҳҜеҗҰжңүдёҖй–Ӣе§Ӣеүөе»әзҡ„е°ҲжЎҲпјҢиӢҘжІ’жңүи«Ӣе…Ҳе°Үе°ҲжЎҲе»әз«Ӣиө·дҫҶдёҰзўәе®ҡе°ҲжЎҲжҳҜеҗҰзӮәгҖҢBlaze PlanгҖҚ

зўәиӘҚе®Ңе°ҲжЎҲеӯҳеңЁеҫҢпјҢйңҖиҰҒйҖІиЎҢеҲқе§ӢеҢ–

ијёе…ҘжҢҮд»ӨеҫҢпјҢжңғеҮәзҸҫиЁұеӨҡе•ҸйЎҢпјҢи«ӢеңЁйҒёж“ҮFirebaseжңҚеӢҷзҡ„е•ҸйЎҢе…§пјҢйҒёж“ҮFunctionsе’ҢRealtime Database(жң¬зҜ„дҫӢжңғзЁҚеҫ®д»Ӣзҙ№еҰӮдҪ•дҪҝз”ЁеңЁCloud FunctionsпјҢдёҚдёҖе®ҡйңҖиҰҒиЈқ)пјҢеҰӮдёӢең–

и«ӢйҒёж“ҮUse an existing projectдёҰйҒёж“Үе·Іеүөе»әзҡ„е°ҲжЎҲпјҢе…¶д»–йҒёй …еүҮжҳҜеҸҜд»ҘзӣҙжҺҘдёҖи·Ҝenterеё¶йҒҺ

йҖҷжЁЈдёҖдҫҶе°ұжҠҠж•ҙеҖӢз’°еўғе»әз«Ӣиө·дҫҶдәҶпјҢи©ізҙ°зҡ„е°ҲжЎҲиЁӯе®ҡд»ҘеҸҠRealtime Databaseзҡ„дҪҝз”Ёж–№ејҸи«ӢжҹҘй–ұе®ҳж–№ж–Үд»¶пјҢз”ұж–јйҖҷдәӣе…§е®№дёҚеңЁжң¬зҜҮиЁҺи«–зҜ„еңҚжүҖд»Ҙе°ұдёҚи©ізҙ°иӘӘжҳҺгҖӮ

е»әз«Ӣе°ҲжЎҲд№ӢеҫҢпјҢеҸҜд»ҘзңӢеҲ°иіҮж–ҷеӨҫе…§жңүдёҖеҖӢfunctionsзҡ„иіҮж–ҷеӨҫпјҢиіҮж–ҷеӨҫе…§жңүдёҖж”ҜеҗҚзӮәindex.jsзҡ„жӘ”жЎҲпјҢйҖҷж”ҜжӘ”жЎҲе°ұжҳҜfunctionж’°еҜ«зҡ„ең°ж–№гҖӮ

й–Ӣе§Ӣж’°еҜ«function

еңЁindex.jsдёӯпјҢжҲ‘еҖ‘еҸҜд»ҘзңӢеҲ°д»–жңүжҸҗдҫӣдёҖдәӣзҜ„дҫӢзЁӢејҸзўјеҰӮдёӢпјҢз•¶дёӯйңҖиҰҒжіЁж„Ҹзҡ„жҳҜзӯҶиҖ…дҪҝз”Ёзҡ„зүҲжң¬жҳҜv2д№ҹе°ұжҳҜз¬¬дәҢд»ЈпјҢи·ҹз¬¬дёҖд»Јзҡ„еҜ«жі•жңғз•ҘжңүдёҚеҗҢгҖӮ

е…¶дёӯжҲ‘еҖ‘жңғзҷјзҸҫеҲ°exports.helloWorldзҡ„е…§е®№е°ұжҳҜе»әз«Ӣfunctionзҡ„ж–№ејҸпјҢд№ҹзӯүж–јд№ӢеҫҢжҲ‘еҖ‘жңғз”ЁдҫҶиӘҝз”Ёзҡ„еҫҢз«ҜAPIпјҢиҖҢзӮәдәҶиғҪеӨ иў«ajaxжүҖиӘҝз”ЁпјҢжҲ‘еҖ‘йңҖиҰҒдҪҝз”ЁonRequestйҖҷеҖӢеҮҪејҸпјҢдҪҝз”Ёж–№жі•еҰӮдёӢ

functionsеӣ зӮәе„ІеӯҳеңЁgoogleзҡ„йӣІз«ҜйҒӢз®—дёӯеҝғпјҢжүҖд»ҘзӮәдәҶеҫ—еҲ°жңҖеҘҪзҡ„йҹҝжҮүйҖҹеәҰпјҢе»әиӯ°иЁӯе®ҡйӣўиҮӘе·ұжҲ–жҳҜе®ўжҲ¶ијғзӮәжҺҘиҝ‘зҡ„иіҮж–ҷдёӯеҝғпјҢзӯҶиҖ…йҖҷиЈЎжҳҜиЁӯе®ҡзӮә asia-east1(еҸ°зҒЈ) пјҢиӢҘжғідҪҝз”Ёе…¶д»–зҡ„иіҮж–ҷдёӯеҝғи«ӢеҸғиҖғгҖӮ
дёҠйқўзҡ„зЁӢејҸиЈЎз°Ўе–®еҜ«дәҶдёҖеҖӢfunctionпјҢеҠҹиғҪжҳҜеҸӘиҰҒеё¶ж•ёеӯ—йҖІдҫҶпјҢе°ұжңғйҡЁж©ҹеҠ дёҠдёҖеҖӢ1-1000зҡ„ж•ёеӯ—дёҰеӣһеӮігҖӮ

еҜ«е®ҢfunctionеҫҢпјҢйңҖиҰҒе°ҮfunctionйғЁзҪІеҲ°йӣІз«ҜпјҢи«ӢдҪҝз”Ёд»ҘдёӢжҢҮд»Ө

йғЁзҪІжҲҗеҠҹд№ӢеҫҢпјҢеӣһеҲ°FirebaseеҫҢеҸ°жңғзҷјзҸҫд»ҘдёӢзҡ„з•«йқўпјҢе…§е®№еҢ…еҗ«дәҶи«ӢжұӮжүҖйңҖзҡ„з¶ІеқҖ

йҖҷеҖӢз¶ІеқҖеҸҜд»ҘдҪҝз”Ё GET гҖҒ POST гҖҒ PUT гҖҒ DELETE е’Ң OPTIONS зҡ„HTTPж–№жі•йҖІиЎҢиӘҝз”ЁпјҢз”ұж–јйҖҷеҖӢfunctionдҪҝз”Ёreq.bodyжҺҘж”¶иіҮж–ҷпјҢжүҖд»ҘйҖҷиЈЎзӯҶиҖ…йҒёз”ЁPOSTзҡ„ж–№ејҸйҖІиЎҢжё¬и©ҰгҖӮ

дҪҝз”ЁPostmanжё¬и©ҰеҸҜд»ҘзҷјзҸҫзўәеҜҰеӣһеӮізөҰжҲ‘йҡЁж©ҹзҡ„ж•ёеӯ—пјҢйҖҷжЁЈе°ұд»ЈиЎЁжҲҗеҠҹдәҶпјҒ

иЈңе……пјҡеҰӮдҪ•жҗӯй…ҚRealtime Database

иӢҘеёҢжңӣеңЁfunctionе…§дҪҝз”ЁRealtime DatabaseеҒҡиіҮж–ҷзҡ„еӯҳеҸ–пјҢеҸҜд»ҘдҪҝз”Ёfirebase-adminдҫҶеҚ”еҠ©е°ҮиіҮж–ҷеҜ«е…ҘиіҮж–ҷеә«

йҖҷиЈЎе°Үе…ҲеүҚзҡ„functionдҝ®ж”№жҲҗжңғе°ҮзөҗжһңеҜ«е…ҘиіҮж–ҷеә«пјҢдё»иҰҒе°ұжҳҜдҪҝз”Ё const db = admin.database(); дҫҶе°ҮиіҮж–ҷеә«еј•е…ҘпјҢеҶҚеҹ·иЎҢеҜ«е…ҘгҖӮ

дҝ®ж”№е®ҢжҲҗеҫҢпјҢеҶҚдҪҝз”ЁPostmanйҖІиЎҢиӘҝз”ЁпјҢжңғзҷјзҸҫиіҮж–ҷеә«зҡ„ж•ёеҖјзўәеҜҰиў«дҝ®ж”№дәҶ

вҖ»жіЁж„Ҹпјҡз”ұж–јFirebaseз¶“еёёеңЁжӣҙж–°пјҢжүҖд»Ҙжңүдәӣе…§е®№еҸҜиғҪжңғе·ІйҒҺжҷӮпјҢиӢҘжңүйҢҜиӘӨйӮ„з…©и«Ӣеҗ„дҪҚжңӢеҸӢе‘ҠзҹҘпјҒ

д»ҘдёҠе°ұжҳҜйҖҷж¬Ўзҡ„е…ЁйғЁе…§е®№пјҢж„ҹи¬қзңӢеҲ°йҖҷиЈЎзҡ„дҪ пјҒ
