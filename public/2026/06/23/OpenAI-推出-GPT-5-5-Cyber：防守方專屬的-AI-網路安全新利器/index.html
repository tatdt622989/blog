OpenAI 推出 GPT-5.5 Cyber：防守方專屬的 AI 網路安全新利器

隨著 AI 代理（Agentic AI）逐漸融入日常開發，AI 攻防戰也正式開打。OpenAI 於 2026 年 6 月發布了專為網路安全防護調校的特化模型：**GPT-5.5 Cyber (New)**。這款模型是 OpenAI 防禦性 AI（Defensive AI）套件的核心，旨在為守方的安全專家提供強大的智慧防禦支援。

什麼是 GPT-5.5 Cyber？

GPT-5.5 Cyber 是基於 GPT-5.5 旗艦模型（代號 Spud）進行安全性微調的特化版本。通用模型在面對敏感的漏洞分析、漏洞利用代碼（Exploit）或攻擊模擬時，常因安全對齊（Safety Alignment）而拒絕回答，或因缺乏網路安全脈絡而僅能給出泛泛建議。

為了突破這個瓶頸，OpenAI 在「Trusted Access for Cyber」計畫下推出了這款特化模型，搭載在 Daybreak 網路安全倡議中。它允許經審核的安全研究人員與防禦者，在受控環境下使用更具彈性的安全存取行為（Permissive access behaviors），以便深入漏洞研究並加速防禦部署。

CyberGym 基準測試數據對比

根據 OpenAI 官方公布的 CyberGym 基準測試數據，最新版的 GPT-5.5-Cyber (New) 展現出極為強悍的性能，擊敗了市場上的主要競爭對手與通用模型：

從 OpenAI 提供的官方數據可以看出，最新版的 GPT-5.5-Cyber (New) 不僅以 85.6% 的得分位居榜首，更顯著超越了通用版 GPT-5.5 的 **81.8%，並大幅領先 Claude Opus 4.7（73.1%**），證明了專門的安全特化訓練對防禦任務有著決定性的提升。

GPT-5.5 Cyber 的三大核心防禦能力

  • 端到端多步驟攻擊模擬：具備在模擬環境中自主執行複雜網路攻擊模擬的能力。它能理解整條攻擊鏈（Kill Chain），並以守方視角逆向分析出攻擊路徑，幫助企業及早堵住潛在漏洞。
  • 漏洞研究與自動化修補：當系統爆發零日漏洞（Zero-day）時，GPT-5.5 Cyber 能快速解析漏洞成因，並自動生成針對該漏洞的修補程式（Patch），將防守效率提升至分鐘級。
  • 關鍵基礎設施防護：模型已提供給歐盟 AI 辦公室（EU AI Office）等政府監管機構。它能在不洩露敏感漏洞細節的前提下，協同各國的網路安全防護中心，針對核心系統進行聯防與早期預警。

如何存取 GPT-5.5 Cyber？

因為具備潛在雙重用途（Dual-use）的特性，本模型不開放給大眾使用。符合資格並通過審查的團隊可以透過以下管道整合：

  1. Trusted Access for Cyber 申請：企業需向 OpenAI 提交安全稽核計畫，通過後方可獲得權限。
  2. API 呼叫：通過審核後，開發者可透過 Responses API 呼叫 gpt-5.5-cyber-preview 模型。
  3. Codex 整合：深度整合於企業內部的 Codex 開發環境中，在撰寫程式碼階段即時進行安全檢測。

結論：防禦性 AI 的未來

GPT-5.5 Cyber 的發布，標誌著網路安全正式進入「用 AI 防禦 AI」的時代。面對自動化、高頻率的 AI 威脅，防禦者唯有依賴性能卓越的特化 Defensive AI，才能在攻防戰中立於不敗之地。
